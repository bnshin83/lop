# Loss of Plasticity in Deep Continual Learning

Reproduction of "Loss of Plasticity in Deep Continual Learning" (Nature 2024).
Paper: https://www.nature.com/articles/s41586-024-07711-7

## Commands
- `/session-start`: Load context
- `/session-end`: Save progress
- `/sync`: Git commit + push

## Setup
- Uses localcontrol for cluster jobs (`.localcontrol/`)
- Main code in `lop/` directory
- Install: `./install.sh` or `pip install -r requirements.txt`

## Key Files
- `README.md`: Full project documentation
- `citations.bib`: Paper citation
- `.localcontrol/`: Cluster experiment configs
